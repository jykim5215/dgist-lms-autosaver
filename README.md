# DGIST LMS AutoSaver

DGIST LMS의 강의 자료를 자동으로 모아 Google Drive에 정리하고,
과제 마감과 학교 이메일까지 한 화면에서 스마트하게 관리하는 데스크톱 앱입니다.

## 주요 기능

- **자료 자동 저장**: LMS 전 과목의 강의 자료·공지 첨부를 자동 다운로드하고 Google Drive(`내 드라이브/AutoSaver/과목명`)에 정리
- **과제 마감 관리**: 과목별 과제 마감일과 내 제출 상태(제출/채점/미제출)를 D-day로 표시, 캘린더(.ics) 내보내기
- **학교 이메일 비서**: 학교 메일을 AI가 분석해 뉴스 카드처럼 정리 — 카테고리 분류, 관심사 기반 추천, 일시·장소·연사 핵심 정보 추출, AI 브리핑
- **빠른 동기화**: 마지막 동기화 이후 변경분만 검사 (수 분 → 수십 초)
- **자동화**: 앱이 켜져 있으면 매일 예약 시간에 자동 동기화 + Windows 알림
- 다크 모드, 한국어 검색, 묶음 다운로드(Shift/Ctrl 다중 선택) 지원

## 설치 (Windows)

1. [Python 3.11 이상](https://www.python.org/downloads/)을 설치합니다.
   설치 화면에서 **"Add Python to PATH"를 꼭 체크**하세요. (한 번만 하면 됩니다)
2. 압축을 풀고 폴더 안의 **`시작하기.bat`** 를 더블클릭합니다.
   - **처음 한 번**: 자동으로 필요한 구성요소를 설치(2~3분)하고 앱을 엽니다.
     설치가 끝나면 **바탕화면에 바로가기 아이콘**이 생깁니다.
   - **그 다음부터**: 바탕화면의 **DGIST LMS AutoSaver** 아이콘만 더블클릭하면 바로 실행됩니다.

> 바로가기는 PC마다 경로가 달라서 압축에 미리 넣을 수 없습니다.
> `시작하기.bat`이 **설치하면서 당신 PC에 맞는 바로가기를 만들어** 줍니다.

## 첫 설정

앱 실행 후 **설정**에서 본인 정보를 입력합니다. 모든 정보는 **내 컴퓨터에만** 저장됩니다 (`C:\lms-autosaver`).

| 항목 | 설명 |
|------|------|
| LMS ID / 비밀번호 | DGIST LMS(Blackboard) 로그인 계정 |
| 구글 계정으로 로그인 | 설정에서 버튼만 누르면 브라우저가 열립니다. 본인 구글 계정으로 로그인해 Drive 권한을 허용하세요. (OAuth 클라이언트 동봉됨) |
| Gemini API Key | 자료 요약·이메일 분석용. [Google AI Studio](https://aistudio.google.com/apikey)에서 무료 발급 |
| 학교 이메일 | DGIST 웹메일 계정 (메일함 기능용) |
| 관심사 | 전공 트랙·관심 활동을 선택하면 AI가 메일을 추천하는 기준이 됩니다 |

설정을 마치면 **지금 동기화**를 눌러 첫 수집을 시작하세요.

## 자주 묻는 질문

- **개인정보는 어디로 가나요?** — 모든 계정 정보·파일은 본인 PC(`C:\lms-autosaver`)에만 저장됩니다. 외부 서버로 전송되지 않습니다.
- **일부 과목 자료가 안 보여요** — 교수님이 콘텐츠 영역을 비공개로 설정한 과목은 API 접근이 막혀 있습니다(공지 첨부는 수집됨).
- **동영상도 받나요?** — 용량 문제로 동영상(mp4 등)은 건너뜁니다.
