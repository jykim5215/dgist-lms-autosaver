DGIST LMS AutoSaver

DGIST LMS AutoSaver는 DGIST 학생이 강의자료와 학교 메일을 자동으로 정리하도록 도와주는 개인용 데스크톱 앱입니다. 사용자의 컴퓨터에서 실행되며, LMS에 올라온 강의자료를 과목별로 내려받고, 과제 마감일을 정리해 보여 주며, 학교 메일에서 찾은 일정을 사용자의 Google 캘린더에 등록하고, 받아 둔 자료를 사용자의 Google 드라이브에 백업합니다.

이 앱이 Google 계정 권한을 요청하는 이유

  • Google 드라이브 — 사용자가 내려받은 강의자료를 사용자 본인의 드라이브에 업로드하기 위해 사용합니다. 앱이 올린 파일에만 접근합니다.
  • Google 캘린더 — 학교 메일에서 추출한 세미나·모집 일정을 앱이 만든 전용 캘린더에 등록하기 위해 사용합니다.

주요 기능

강의자료 자동 저장

LMS에 올라온 자료를 과목별 폴더로 내려받아 정리합니다. 과목별로 포함·제외를 직접 고를 수 있습니다.

과제 마감 관리

남은 기한을 D-day로 보여 주고, 제출 여부까지 함께 표시합니다.

메일함과 브리핑

학교 메일을 읽어 관심사에 맞게 분류하고, 중요한 것만 짧게 요약합니다.

구글 캘린더 동기화

메일에서 찾은 세미나·모집 일정을 전용 캘린더에 자동으로 등록합니다.

구글 드라이브 백업

받아 둔 자료를 내 드라이브의 앱 전용 폴더에 올려 둘 수 있습니다.

자동 업데이트

새 버전이 올라오면 앱 안에서 바로 최신으로 갱신합니다.

내 데이터는 어디에 있나요

이 앱에는 개발자가 운영하는 서버가 없습니다. 앱은 사용자 컴퓨터에서 실행되고, 아이디·비밀번호·받은 자료·메일은 모두 그 컴퓨터 안에만 저장됩니다. 개발자는 사용자의 데이터를 수집하거나 열람하지 않습니다.

구글 드라이브·캘린더·Gemini 같은 외부 서비스는 사용자가 직접 연결했을 때만, 해당 기능에 필요한 범위에서만 사용됩니다. 자세한 내용은 개인정보처리방침에 정리해 두었습니다.
