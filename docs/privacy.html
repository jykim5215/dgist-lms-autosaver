최종 수정일: 2026년 7월 23일 · 적용 대상: DGIST LMS AutoSaver 앱(이하 “앱”)

1. 핵심 요약

  • 개발자가 운영하는 서버가 없습니다. 앱은 전적으로 사용자 본인의 컴퓨터에서 실행됩니다.
  • 개발자는 사용자의 계정 정보, 강의자료, 메일, 일정을 수집·저장·열람하지 않습니다.
  • 모든 자격증명과 데이터는 사용자 컴퓨터 내부에만 저장됩니다.
  • 외부 서비스로의 전송은 사용자가 그 기능을 직접 켰을 때, 해당 기능 수행에 필요한 범위에서만 일어납니다.
  • 앱을 지우면 관련 데이터도 함께 사라집니다. 개발자 측에 남는 사본이 없습니다.

3. 외부로 전송되는 정보

앱은 다음 서비스와 통신합니다. 각 항목은 해당 기능을 사용할 때만 동작합니다.

DGIST LMS

로그인 정보를 학교 LMS에 전송하여 강의자료와 과제 정보를 가져옵니다.

학교 메일 서버 (IMAP·SMTP)

메일을 읽거나 보내기 위해 메일 서버에 접속합니다. 메일 내용은 사용자와 메일 서버 사이에서만 오갑니다.

Google Drive

사용자가 업로드 기능을 켠 경우, 내려받은 강의자료를 사용자 본인의 드라이브에 올립니다. drive.file 권한만 사용하므로 앱이 만들거나 올린 파일에만 접근하며, 드라이브의 다른 파일은 읽을 수 없습니다.

Google Calendar

사용자가 캘린더 동기화를 켠 경우, 메일에서 추출한 일정 제목·날짜·발신자 정보를 사용자 캘린더에 등록합니다. 앱은 전용 캘린더(기본 이름 “DGIST 메일 일정”)를 직접 만들고 그 캘린더에만 일정을 쓰며, 기존 일정을 수정하거나 삭제하지 않습니다.

Google Gemini API

메일 요약·분류 기능을 사용할 경우, 메일의 제목과 본문이 Google Gemini API로 전송됩니다. 이 기능이 필요하지 않다면 설정에서 Gemini API 키를 비워 두어 사용하지 않을 수 있습니다.

GitHub

앱 업데이트 확인 및 최신 파일 내려받기에만 사용되며, 사용자 데이터는 전송되지 않습니다.

4. Google 사용자 데이터의 사용 제한

이 앱이 Google API로부터 받은 정보의 사용 및 다른 앱으로의 전송은 Google API 서비스 사용자 데이터 정책 (제한적 사용 요건 포함)을 준수합니다. 구체적으로 앱은 Google 사용자 데이터를 사용자에게 보이는 기능을 제공하는 목적으로만 사용하며, 광고 목적으로 사용하거나 제3자에게 판매·양도하지 않고, 사람이 열람하지 않습니다.

5. 보관 기간과 삭제

  • 데이터는 사용자가 직접 지울 때까지 사용자 컴퓨터에 남습니다.
  • 앱 폴더와 데이터 폴더를 삭제하면 저장된 정보가 함께 삭제됩니다.
  • 구글 계정 연결은 앱의 설정 → 구글 로그인 해제에서 끊을 수 있습니다.
  • 구글 계정 쪽에서 직접 권한을 철회하려면 Google 계정 > 타사 앱 및 서비스에서 이 앱의 액세스를 삭제하면 됩니다.

6. 보안

자격증명은 사용자 컴퓨터의 설정 파일에 저장되므로, 해당 컴퓨터의 보안 수준이 곧 데이터의 보안 수준입니다. 공용 컴퓨터에서는 사용을 권하지 않으며, 사용 후에는 반드시 계정 연결을 해제해 주세요. 구글 인증에는 비밀번호 대신 OAuth 토큰을 사용하므로 앱이 구글 비밀번호를 알지 못합니다.

7. 아동의 개인정보

이 앱은 DGIST 재학생을 위한 도구이며, 만 14세 미만 아동을 대상으로 하지 않습니다.
